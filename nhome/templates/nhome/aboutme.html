This is a front end interface for an already existing localhost webserver to run nmap command with the click of a button for non-technical staff


How it works:



The user needs to have Nmap installed, and Apache Webserver running on localhost.
The webserver calls the shellscript which executes nmap on the user's host and subnetwork.
Nmap will then execute the command provided by the user and return the results (e.g port scans of Host 2 and Host 3)

Pre-Usage Configuration on host:

Step 1:
Install Apache (If already not installed)
sudo apt-get install apache2


Step 2:
Start Apache2
Sudo systemctl apache2 start


Step 3:
Install Nmap (if not already installed)
sudo apt-get install nmap


Step 4:
Download given code and place in cd /usr/lib/cgi-bin
and run sudo chmod a+x nmapp.sh
